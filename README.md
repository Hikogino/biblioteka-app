# Aplikacja internetowa do zarządzania kolekcją książek z systemem śledzenia postępu i celów czytelniczych

Projekt dyplomowy z informatyki — aplikacja umożliwia śledzenie historii czytania, analizę nawyków czytelniczych oraz realizację spersonalizowanych celów czytelniczych.

## Funkcjonalności

- **Czytnik** — wbudowany czytnik obsługujący formaty FB2, EPUB i PDF
- **Cele** — system celów czytelniczych z prognozowaniem wyników
- **Analityka** — wykresy aktywności, tempo czytania, porównanie okresów
- **Zakładki i notatki** — zapisywanie miejsc i cytatów podczas czytania
- **Konta użytkowników** — rejestracja, logowanie, dane przypisane do konta

## Technologie

- **Backend:** Node.js, Express.js
- **Baza danych:** MongoDB, Mongoose
- **Frontend:** HTML5, CSS3, JavaScript ES6+
- **Biblioteki:** Chart.js, JSZip, bcryptjs, Multer, express-session

## Struktura projektu

```
├── controllers/    # Logika biznesowa
├── models/         # Modele MongoDB
├── routes/         # Routing Express
├── views/          # Szablony EJS
├── public/         # Pliki statyczne (CSS, JS, obrazy)
└── middleware/     # Middleware autoryzacji
```
